# WP Announcements — Анонсы книг

Плагин WordPress/WooCommerce для работы с «анонсами» — товарами, которые пока нельзя купить, но можно оформить предзаказ и получить уведомление по email, когда товар появится в обычном каталоге.

## Возможности

- Выбор одной или нескольких product_cat как «анонсных» (в настройках плагина).
- Товары из анонсных категорий автоматически:
  - становятся непокупаемыми (`is_purchasable = false`);
  - показывают кнопку **«Предзаказ»** вместо «В корзину» — и в каталоге, и в карточке товара.
- При клике на «Предзаказ» открывается модалка с полями: имя, email (обязательно), телефон.
- Подписки пишутся в таблицу `wp_wpan_subs` (админке доступен список).
- Когда товар выводится из анонсной категории (вручную или через импорт WooMS) — всем подписчикам автоматически уходит письмо о поступлении; админу приходит отчёт.
- Шорткоды:
  - `[announcements_grid limit="8" columns="4" title="Скоро в продаже"]` — сетка для главной.
  - `[announcements_page per_page="24" columns="4"]` — отдельная страница со всеми анонсами и пагинацией.

## Установка

1. Скопировать папку `wp-announcements/` в `wp-content/plugins/`.
2. Активировать в админке (создастся таблица `wp_wpan_subs`).
3. WooCommerce → **Анонсы → Настройки** — выбрать категории, настроить тексты и шаблоны писем.

## Как это работает

- При сохранении товара (`woocommerce_update_product`, `woocommerce_new_product`) и при изменении таксономии (`set_object_terms`) плагин пересчитывает мета-флаг `_wpan_is_announcement`.
- Если товар был анонсом, а после сохранения таковым быть перестал — вызывается `wpan_notify_subscribers($product_id)`:
  - отправляется письмо каждому подписчику из `wp_wpan_subs`, где `notified_at IS NULL`;
  - после успешной отправки проставляется `notified_at`;
  - админу уходит итоговое письмо с количеством получателей.

## Интеграция с WooMS

Синхронизация из МойСклад (плагин `wooms`) меняет категорию товара через стандартные WooCommerce API, поэтому ничего специально интегрировать не нужно — перенос книги из анонсной папки в обычную автоматически запускает рассылку.

## Переменные в шаблонах писем

- `{product}` — название товара
- `{product_url}` — ссылка на товар
- `{site_name}` — название сайта
- `{count}` — кол-во уведомлённых (только в админском письме о рассылке)
