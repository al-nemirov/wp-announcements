# Changelog

## 1.2.0
- **Главная настройка теперь — «Пути папок в МойСклад»** (textarea). Сверяется как подстрока с полем `pathName`, которое wooms передаёт из МС. Товар = анонс, если его МС-путь содержит любую из указанных строк — независимо от WP-категории.
- В настройках показывается список уже обнаруженных `pathName` на товарах (клик по строке — копирование в буфер).
- Хук `wooms_product_update` (priority 100) сохраняет `pathName` в post_meta `_wpan_ms_path` при каждой синхронизации.
- WP-категории оставлены как «запасной вариант» для сайтов без МойСклад.
- Единая функция `wpan_queue_notify()` и упрощение логики `set_object_terms`.

## 1.1.1
- Плагин сделан универсальным — убраны упоминания книг в названии, описании, шаблонах писем.
- Настройки карточки: произвольный текст бейджа, галочки «показывать метки» и «показывать цену».
- README переписан под универсальный сценарий.

## 1.1.0

### Критичные исправления
- `set_object_terms` теперь читает актуальный набор категорий через `wp_get_object_terms`, корректно работает при `append=true`.
- Замена «В корзину» → «Предзаказ» теперь работает и через `woocommerce_product_add_to_cart_text` / `woocommerce_product_add_to_cart_url` — ловим темы с кастомным лупом.
- Email нормализуется в lowercase перед поиском/вставкой — нет дублей подписок.

### Новое
- Удалено поле «телефон» везде (БД, форма, админка, письма). Автомиграция схемы БД при активации/обновлении.
- Honeypot-поле `website` + rate-limit по IP/час (настраивается, 20 по умолчанию).
- Очередь рассылки через WP-Cron (`hourly` + single-event через 30 секунд после триггера) — не бьёт лимит SMTP при bulk-операциях.
- CSV-экспорт подписок (с BOM для Excel).
- Пагинация в списке подписок (50 на страницу).
- Текст «Уже подписаны» — настраиваемый.
- Цвет кнопки и hover — через настройки (CSS-переменные `--wpan-color`, `--wpan-color-hover`).
- Автор в карточке: показываются **все** метки через запятую (а не только первая).
- Orderby в шорткодах — whitelist (`date`, `title`, `menu_order`, `rand`, `modified`, `ID`).
- Модалка: `max-height:92vh` + `overflow-y:auto` — корректная работа на мобиле.
- `uninstall.php` — опциональная очистка таблицы/опций при удалении (галочка в настройках).
- Загрузка текст-домена для i18n.
- Убран дублирующий подпункт меню «Настройки».

### Внутренне
- Static-cache для `get_option(WPAN_OPT)` — меньше запросов в БД на странице с сеткой.
- `wpan_default_settings()` — единое место дефолтов.
